# node-alias-windows

A quick and dirty re implementation of alias command for windows system.

## install

```npm i node-alias-windows -g```

## usage

```bash

    aliasw ls "dir"
    aliasw whatever "some\"more complex/\"command"
    

```

## note

not yet tested on windows, but i expect it to work smoothly.
